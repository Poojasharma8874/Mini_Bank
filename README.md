# Bank Management System (Console Based | Java | OOPs)
This is a **console-based Bank Management System** built using **Core Java and OOPs concepts**.
It includes:
- BankAccount Class  
- Main Class (bank operations)  
- Encapsulation, Constructors, Methods  
- Basic operations like deposit, withdraw, check balance

# Features
✔ Create a new bank account  
✔ Deposit money  
✔ Withdraw money  
✔ Check account balance  
✔ Exit the system

# Tech Stack
- Java  
- OOPs (Encapsulation, Classes & Objects)  
- Eclipse
